== Installation ==
This plugin follows the [standard WordPress installation method][]:

1. Go to the <strong>Plugins > Add New page</strong> in your WordPress admin.
2. Search for "JetPush" and install <strong>JetPush Embeded Code</strong>.
3. Click <strong>Activate Plugin</strong>.
4. Go to <strong>Settings > JetPush Plugin Setting</strong> and enter your JetPush ID there. An example of JetPush ID --> <code>m69h8semi</code>.
5. Save it & your done.

[standard WordPress installation method]: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
